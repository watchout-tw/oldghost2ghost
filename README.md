# oldghost2ghost

To move all the `GHOST Articles` from old version to new version

## System Requirement

node version up to 8

## Installation

use npm

``` bash
npm install
```

use yarn

``` bash
yarn
```

## ENV File Setup

copy `.env.sample` file and rename to `.env`

## Usage

``` bash
npm run develop
```

or

``` bash
yarn develop
```
